Recreating the mojo website

Informatie

Mijn rol: Front end developer

Individuele opdracht

Duur: periode van 3 weken

Vak: Front end development, leerjaar 2 CMD

Opdracht

Voor het vak front end development moesten wij een bestaande website zo goed mogelijk proberen na te maken. Waarvoor ik de website van Mojo had gekozen.

Alle content die ik hiervoor heb gebruikt komt van de site Mojo en is dus geen eigen content geweest.

Het ging er dan ook om tijdens dit project dat wij coderen meer onder de knie kregen en meer met verschillende selectoren aan de slag konden gaan.

Daarnaast hebben wij tijdens dit vak gebruik gemaakt van de screenreader en leren begrijpen wat het belang is van toegankelijkheid in een website.

Doel

Het doel was om twee verschillende pagina's uit te werken uit nette html, css en JavaScript. Met daarbij een extra uitwerking tot een responsive design.

Proces

We hebben tijdens dit project veel gefocust op het onder de knie krijgen van de code. Daarbij hebben wij gebruik gemaakt van een breakdownschets en het maken van opdrachten via codepen.

Breakdown schets

Voor dit project hebben wij gebruik gemaakt van een breakdow schets die wij zelf moesten maken van de site die wij wilden namaken.

Oefenen met CodePen

We hebben tijden dit project ook veel gebruik gemaakt van opdrachten die wij hebben gemaakt via CodePen, dit vond ik een handige tool om te gebruiken, omdat je er met het oefenen van coderen makkelijk kan zien wat bepaalde code doet.

Zo hebben we hiernaast een animatie gemaakt door een class te geven via JavaScript

Het lastigste vond ik tijdens dit project dat zodra er een fout in de code staat dat je soms langer bezig bent om deze fout te vinden, maar gelukkig bestaan hier tegenwoordig steeds meer tools voor die je daarvoor kan gebruiken.
